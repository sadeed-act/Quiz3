Q2: What will be the output of the following code?

php
class Car { 
  public $model; 
  public function __construct($model) { 
    $this->model = $model; 
  } 
} 
$myCar = new Car("Toyota"); 
echo $myCar->model;
Correct Answer: Toyota

Q3: Write PHP statement establishes a procedural connection to a MySQL database, assuming the server is localhost, username is web_user, password is secure_pass, and database name is app_db, and stores the connection in a variable named $conn?

Correct Answer: $conn = mysqli_connect("localhost", "web_user", "secure_pass", "app_db");

Q7: Look at the following query:

SELECT products.name, categories.name
FROM categories
RIGHT JOIN products ON categories.id = products.category_id;

Which table is considered the "right" table in this join?

Correct Answer: C. products

Q8: Write an SQL SELECT query to get the name and email of all users from a table named users who were created after June 1, 2024. Order the results by name ascending order (A-Z).

Correct Answer:
SELECT name, email FROM users WHERE created_at > '2024-06-01' ORDER BY name ASC;
